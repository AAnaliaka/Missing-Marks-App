
               MMUST MISSING MARKS APP

1. Splash Screen
•	University logo
•	App name: "Missing Marks Tracker"
•	Optional tagline: "Stay on top of your academic performance"
2. Login Page
•	Fields: 
o	Username/Email
o	Password
•	Buttons: 
o	Login
o	Forgot Password?
o	Sign Up (Redirect to registration page)

3. Home Page (Dashboard)
•	For Students:
o	Overview of current courses
o	Missing marks summary (e.g., total assignments missed)
o	Important announcements/notifications
o	Quick links to reports and grades
•	Home Page (Dashboard)
•	For Lecturers:
o	Overview of courses taught
o	Missing marks summary for each student
o	Notifications on submitted assignments
4. Courses Page
•	List of enrolled courses for students
•	List of courses taught for lecturers
•	Option to click on a course for more details
5. Assignments Page
•	For Students:
o	List of assignments with due dates
o	Status (Submitted/Not Submitted)
o	Missing assignments highlighted
•	For Lecturers:
o	List of assignments created
o	Option to add/edit assignments
o	View submission statuses
6. Missing Marks Report Page
•	For Students:
o	Detailed view of missing assignments
o	Option to submit late work (if allowed)
o	Contact lecturer button for queries
•	For Lecturers:
o	Student-wise missing marks report
o	Options to record marks/feedback
o	Export report as PDF
7. Profile Page
•	User information (name, email, profile picture)
•	Option to update personal information
•	Change password option
8. Notifications Page
•	List of alerts (e.g., assignments due soon, lecturer messages)
•	Option to mark notifications as read/unread
9. Settings Page
•	Notification preferences
•	Theme selection (light/dark mode)
•	Help & support section
•	Logout button
10. Help Page
•	FAQs
•	Contact support (email or chat)
•	User guide/instructions for using the app
